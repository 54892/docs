# Access Tokens

Access tokens can be your API or Personal Tokens managed inside your [account settings](/docs/usage/settings#token).

## Creating API Token

API tokens allow you to access one or more datasets for either querying data, ingesting data, or both.

To create an API token, navigate to **Settings --> API Tokens** and click the **Add API Token** button.

![API Token Overview](/doc-assets/shots/api-token.png)

For security reasons it is recommended to use an API Token with minimal privileges only (Querying and Ingesting).

See [API Token](/docs/reference/settings#token) for more detail.

## Creating Personal Token

Personal tokens allow you to access the Axiom API programmatically for custom integrations. Personal token grants access to all resources available to the user.

To create a Personal token, navigate to **Settings --> Profile** and click the **Add Personal Token** button.

![Personal Token Overview](/doc-assets/shots/personal-token.png)

See [Personal Token](/docs/reference/settings#token) for more detail.
