# Sending Events using API clients

Send data directly to Axiom API using our libraries. Currently, we have the following API clients for your convenience:

- [Go Client](https://github.com/axiomhq/axiom-go)

- [JS Client](https://github.com/axiomhq/axiom-js)

- [Next-Axiom](https://github.com/axiomhq/next-axiom)

- [Python Client](https://github.com/axiomhq/axiom-py)

- [Rust Client](https://github.com/axiomhq/axiom-rs)
