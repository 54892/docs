---
dataset: [{
  service: "api",
  log: '{ "message": "Failed to connect to database", "level": "error" }'
}, {
  service: "api",
  log: '{ "message": "Failed to update user", "level": "error" }'
}, {
  service: "frontend",
  log: '{ "message": "Exception on login", "level": "warn" }',
  path: '/'
}, {
  service: "api",
  log: '{ "message": "Migrated database", "level": "info" }'
}, {
  service: "frontend",
  log: null,
  path: '/login'
}]
query: |-
  example
  | where isnotnull(log)
  | extend parsed_log = parse_json(log)
  | project service, parsed_log.level, parsed_log.message
---

The `parse_json` function allows you to parse a JSON string.
