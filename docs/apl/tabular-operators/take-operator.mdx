# take operator

Returns up to the specified number of rows.


## Syntax

```apl copy
| take
```

The [limit operator](/docs/apl/tabular-operators/limit-operator) is an alias to the take operator. 

| **name**     | **type**  | **description**                                                       |
| ------------ | --------- | --------------------------------------------------------------------- |
| Number Of Rows | **integer** | The number of rows to return. You can specify any numeric expression. |

## Examples

```apl copy
['http-logs']
| take 1000
```
